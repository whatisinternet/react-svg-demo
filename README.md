# SVG demo app for [SVG to React Coffee](https://github.com/whatisinternet/svg_to_react_coffee)

A demo of the SVG to react converter.

-----

## Install and run:

    npm install && npm run app

## See results:

  [localhost:8080](http://localhost:8080)
